# RD Tattoo Analysis

A Python package for analyzing and visualizing tattoo patterns using reaction-diffusion models.

## Installation

```bash
pip install -e .
```

## Usage

The package can be used in Jupyter notebooks or as a Python module. See the example notebooks:
- `tattoo.ipynb`
- `tattoo2.ipynb`

## Development

This package is installed in development mode using:
```bash
pip install -e .
```
